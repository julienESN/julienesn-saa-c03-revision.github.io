# SAA-C03 · Console de révision

Site de révision pour la certification **AWS Certified Solutions Architect – Associate (SAA-C03)**, construit à partir de mes notes manuscrites (cours Stéphane Maarek) avec une approche **exam-first**.

**🔗 Live : https://julienESN.github.io/saa-c03-revision/**

## Approche

Chaque service se lit en 3 couches :

1. **Notes** — comprendre le service (synthèse de mes notes de cours)
2. **Déclencheurs exam** — le format réel des questions SAA : *scénario → réponse* (73 réflexes)
3. **Réflexe FinOps** — l'angle coût/optimisation pour chaque service, mon axe de spécialisation

## Fonctionnalités

- **Fiches** : 14 services (IAM, EC2, EBS, EFS, S3, ELB, ASG, RDS, Aurora, ElastiCache, Route 53…) avec scroll-spy
- **Réflexes exam** : les 73 déclencheurs en un tableau de bord filtrable (révision de dernière minute)
- **Rappel actif** : flashcards mélangées avec navigation clavier — la méthode de mémorisation par récupération

## Stack

HTML/CSS/JS vanilla, un seul fichier, zéro dépendance, fonctionne hors ligne.
